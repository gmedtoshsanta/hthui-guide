# hthui-guide

## 项目简介

hthui-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面可能涉及各类技术演示、原型设计、学习笔记或轻量级工具界面，不针对任何特定域名或网站。所有页面均为静态 HTML，可直接通过浏览器访问。

## 目录说明

```
hthui-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，命名清晰以便识别。
- `assets/`：公共资源可在此目录下按类型或页面分类存放。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间没有依赖关系。每个页面均可单独打开并运行，部分页面可能引用 `assets/` 下的共享资源。

页面内容可能包括但不限于：
- 前端组件或布局的演示
- 技术概念的可视化或交互式说明
- 个人项目或学习过程中的临时产出

页面不包含任何商业推广、外部链接跳转或用户数据收集功能。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会直接提交至对应目录。
- 欢迎通过 Issues 提交建议或指出页面中的问题。
- 暂不接受 Pull Request 贡献，但 Issues 中的反馈会被认真考虑。
- 仓库可能随时调整目录结构或清理老旧页面，恕不另行通知。

## 使用方式

1. 克隆或下载本仓库。
2. 使用浏览器直接打开 `pages/` 目录下的 HTML 文件。
3. 也可通过 GitHub Pages 或其他静态托管服务部署后浏览（需自行配置）。

## 许可

本仓库内容仅供学习和参考使用，未明确声明许可协议。如有特殊需求，请通过 Issues 联系维护者。
